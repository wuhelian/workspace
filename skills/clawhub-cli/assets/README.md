# Assets

This folder contains **reusable, non-code** assets for the `clawhub-cli` skill.

Recommended contents:

- `skill-icon.svg`: a small, text-based icon that can be reused in docs.
- `skill-card-template.md`: a markdown template to document skills consistently.

Keep assets lightweight. Prefer text formats (`.svg`, `.md`, `.json`, `.yaml`) over large binaries.
